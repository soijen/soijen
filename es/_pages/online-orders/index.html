Pedidos Online

Ahora aceptamos pedidos online y despachos SOLO dentro de Estados Unidos.

Hay dos formas de hacer el pedido:

    1. Mirar nuestros diseños y completar el formulario. Responderemos con un enlace para el pago via Paypal o Venmo. Una vez hecho el pago, nosotros enviaremos el producto. ¡Ofrecemos despacho gratis, si las ordenes son realizadas via este medio!
    2. Ordenar via nuestra Tienda de Etsy.
